Sobre

Este projeto recupera uma lista das modificações da semana usando a Wikimedia Tool Labs, e envia os emails usando o Mailchimp. Código e mais informações disponíveis no GitHub GitHub. Feito por Stephen LaPorte e Mahmoud Hashemi.
